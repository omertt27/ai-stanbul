## Key Integration Points

1. **MapResponseIntegrator** (NEW component)
   - Bridges LLM system and map visualization
   - Handles single and multi-intent map generation
   - Intelligently combines maps

2. **Multi-Intent Handler** (UPDATED)
   - Calls MapResponseIntegrator after intent execution
   - Passes map_data to response synthesis

3. **Response Synthesizer** (UPDATED)
   - Accepts map_data parameter
   - Informs LLM about map availability
   - Includes map_data in result

4. **Chat API** (UPDATED)
   - Extracts map_data from synthesis result
   - Returns in ChatResponse
